# 📂 VisionX-OS Tools Manifest

VisionX-OS comes bundled with a curated set of tools, organized into modules for clarity and speed. Each module groups related utilities, helping you focus on the task at hand without the need for manual configuration.

---

## 🛠️ Global CLI Interface

Accessing and managing tools in VisionX-OS is streamlined through the `visionx` command-line utility.

| Command | Action |
| :--- | :--- |
| `visionx tools list` | View all installed packages and their current versions. |
| `visionx help <tool>` | Access guided, beginner-friendly documentation for any utility. |
| `visionx update` | Synchronize your local toolset with the official VisionX repositories. |
